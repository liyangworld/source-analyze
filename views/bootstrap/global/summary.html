全局样式 - 概述

Bootstrap 的源码是基于 CSS 预处理语言 Less 开发的，同时维护了 Sass 版。其后将以 Less 为基础对 Bootstrap v3.3.7 进行梳理。

bootstrap.less

bootstrap.less 文件主要是导入其他的模块化文件。

  • Less
  • Sass
